# Code of conduct

This project adopts the
**[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

All contributors, maintainers, and participants in project spaces are expected to follow
it. The full text — including scope, enforcement guidelines, and definitions — lives at
the canonical URL above and is not duplicated here so this file remains in sync with any
clarifications the upstream publishes.

## Reporting

Concerns about conduct should be sent privately to **peter.seliger@googlemail.com**.

Reports will be acknowledged within 72 hours. The maintainers will review and respond as
outlined in the "Enforcement Responsibilities" section of the Contributor Covenant.
