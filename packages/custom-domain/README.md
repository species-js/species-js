# @species-js/custom-domain

Customizable prototype-less namespace objects for sealed method grouping.

Part of the [species-js](https://github.com/species-js/species-js) monorepo — foundation
runtime type infrastructure.

## Install

```sh
pnpm add @species-js/custom-domain
```

## Runtime support

Chrome 80+, Firefox 74+, Safari 13.1+, Edge 80+, Node 22+. See the `browserslist` field in
`package.json`.
