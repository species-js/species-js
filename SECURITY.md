# Security policy

## Supported versions

Only the latest minor release on each major line receives security updates.

## Reporting a vulnerability

Please **do not** open public issues for security vulnerabilities.

Report security issues privately via GitHub Security Advisories:

→
[Report a vulnerability](https://github.com/species-js/species-js/security/advisories/new)

We will acknowledge receipt within 72 hours and aim to provide a triage update within
seven days. Coordinated disclosure timelines will be agreed with the reporter.
